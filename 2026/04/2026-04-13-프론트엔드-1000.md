## 모범 답변

React의 Virtual DOM은 상태 변경 시 React가 새로운 Virtual DOM을 생성하고, 이전 Virtual DOM과 비교하여 변경된 부분만 실제 DOM에 반영하는 방식입니다. 이러한 diffing 과정은 효율적인 reconciliation을 통해 이루어지며, 실제 DOM 조작을 최소화하여 UI 업데이트를 빠르고 효율적으로 처리할 수 있습니다. 예를 들어, 이러한 방식으로 React는 UI 업데이트를 50% 더 빠르게 처리할 수 있습니다.
